# Tab, Tab, Attack

## Challenge Information

- **Category**: General Skills
- **Difficulty**: Easy
- **Author**: syreal

## Description

Using tabcomplete in the Terminal will add years to your life, esp. when dealing with long rambling directory structures and filenames.

## Solution

### Step 1: Extract the archive

```bash
unzip Addadshashanammu.zip
```

### Step 2: Navigate to the final file

The challenge is solved using **tab completion** in the terminal. After unzipping, the path to the flag is deeply nested:

```
Addadshashanammu/Almurbalarammi/Ashalmimilkala/Assurnabitashpi/Maelkashishi/Onnissiralis/Ularradallaku/fang-of-haynekhtnamet
```

### Step 3: Read the flag

Run the binary or read the file to get the flag:

```
*ZAP!* picoCTF{l3v3l_up!_t4k3_4_r35t!_fc588427}
```
